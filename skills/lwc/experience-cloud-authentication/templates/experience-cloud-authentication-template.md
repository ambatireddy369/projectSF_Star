## Review Checklist

- [ ] Auth provider callback URL registered with IdP for sandbox
- [ ] Auth provider callback URL registered with IdP for production
- [ ] Registration handler implements correct interface for flow type
- [ ] Custom login page type matches site engine (VF for Aura, LWC for LWR)
- [ ] LWC login page deployed with `lightningCommunity__Page` target before assigning
- [ ] FederationIdentifier populated and matched correctly
- [ ] Headless Identity enabled in org if using headless flow
- [ ] Both `HeadlessUserDiscoveryHandler` and `HeadlessSelfRegistrationHandler` assigned if using headless flow
- [ ] Sandbox end-to-end login test completed (full IdP round-trip)
- [ ] Login History in Setup reviewed to confirm successful authentications
